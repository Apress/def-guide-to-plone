This is a collection of examples from the Plone book.

Installing:

- Copy the whole directory containing this file into the Products directory of your Plone instance

- Restart your Plone.

- Log in to your Plone site and go to portal setup > Add / Remove Products

- Then click on install PloneBookExamples

This is the procedure documented in Chapter 10 of the book.

You should then have the scripts and forms as documented in the book, without having the tedious retyping. You can call all the templates through the browser by calling the appropriate template:

    eg: http://yoursite.com/Plone/testForm
